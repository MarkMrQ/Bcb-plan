The clever bit

Intent × geo, two systems

Intent — what "best" means · content + UX
Sort, don't guess
  1. Homepage asks: biggest, least hassle, no deposit.
  2. Routes to the matching cluster page.
  3. Sortable Real Value Score: size, wagering, payout speed.
Geo — where they are · edge, at runtime
Nudge, don't redirect
  1. Edge-detect country (Cloudflare header).
  2. Dismissible nudge, not a redirect.
  3. Pre-sort locale offers, persist choice, manual override.
Why no auto-redirect. Googlebot crawls mostly from US IPs, so an IP redirect wrecks indexing, and VPNs make detection unreliable. Same canonical URL for crawler and every user; the signal suggests, never forces. The firewall is structural: the grey arm is a separate site, so geo detection is never the compliance gate.
